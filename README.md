# PandaCoding — Windows Installer

**Claude Code, 7x cheaper.**

PandaCoding gives you full access to Claude Code at a fraction of the official price.

## Download

Go to the [Releases](../../releases) page and download the latest `ClawKit-*.zip`.

## Install

1. Download `ClawKit-x.x.x-win.zip`
2. Extract anywhere
3. Run `PandaCoding Setup x.x.x.exe`
4. Follow the installer (enter your API key when prompted)

## Requirements

- Windows 10 / 11 (64-bit)
- An API key from [pandacoding.cloud](https://pandacoding.cloud)

## Get an API Key

Sign up at **[pandacoding.cloud](https://pandacoding.cloud)** to get your API key.
