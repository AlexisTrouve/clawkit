# PandaCoding

> Claude Code at 7x less than the official price.

PandaCoding lets you run [Claude Code](https://claude.ai/code) — Anthropic's AI coding assistant — through our proxy, without paying for a Claude Max subscription.

## What you get

- Full Claude Code experience in your terminal
- Access to Claude Opus 4, Sonnet 4, and Haiku
- Pay per token — no monthly commitment
- Windows installer, zero setup headache

## Download

**[→ Download ClawKit-1.0.0-win-en.zip](https://pandacoding.cloud/downloads/ClawKit-1.0.0-win-en.zip)** (474 MB)

## Installation

1. Download and extract the zip
2. Run **`PandaCoding Setup 1.0.0.exe`**
3. Enter your API key when prompted
4. Done — Claude Code opens in your terminal via `claude`

The installer sets up a local proxy that runs silently in the background. No manual config needed.

## Requirements

- Windows 10 or 11 (64-bit)
- A PandaCoding API key — get one at [pandacoding.cloud](https://pandacoding.cloud)

## FAQ

**Does it work like the official Claude Code?**
Yes. It runs the exact same OpenClaw client, pointed at our proxy instead of Anthropic directly.

**What happens to my conversations?**
Nothing is stored on our end. Your sessions go through our proxy and straight to the AI — we only see token counts for billing.

**Can I use it alongside other tools?**
Yes. The proxy runs on `localhost:3456` and is compatible with anything that accepts an OpenAI-compatible API endpoint.

**Something broke — where do I get help?**
→ support@pandacoding.cloud

## Uninstall

Open **Add or remove programs** → search PandaCoding → Uninstall.
Or run `uninstall.exe` in your installation folder.
